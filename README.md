# Bright Web D 31:8 — Demo Site

This is a small demo landing site for the brand "Bright Web D 31:8" built with Create React App.

To run locally:

```bash
cd my-react-app
npm install
npm start
```

Open http://localhost:3000 in your browser.

Additional dependency used for routing:

```bash
cd my-react-app
npm install react-router-dom
```

After installing dependencies, run `npm start`.
